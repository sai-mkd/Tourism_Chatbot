Smart Tourism Chatbot

An interactive Chatbot developed using RAG, Python, FAISS that guides the users on their vacations to new destinations

Features
1. Communicates with user interactively
2. Provides list of available hotels and restaurants
3. Informs about the tourist hotspots in the destination
4. Learns about user and provides curated responses

Future Enhancements
1. Navigation integration
2. Improving responses
